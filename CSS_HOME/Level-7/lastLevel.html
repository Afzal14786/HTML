Pseudo Classes

  • Used to define special states of HTML elements.
  • Syntax : selector : pseudo-class{style}.
  • Comman examples: ":hover", ":active", ":first-child".
  • Target elements based on their position or user action.

Transitions

CSS transition is a property that enables smooth animation between changes in CSS propeerty values.

  • transition-property : Defines which CSS properties will transition.
  • transition-duration : Set how long the transition lasts.
  • transition-timing-function : Conrol the speed curve of the transition.
  • transition-delay : Specifies a delay before the transition starts.

CSS Transform

  • Allow modification of an element's shape and position.
  • Can perform operation like : "rotate", "scale", and "translate".
  • Does not affact the layout of surrounding elements.
  • Used to create visual effect like "3D" space transformation.
  • Implemented with functions like "rotate()", "scale()", and "translate()"
