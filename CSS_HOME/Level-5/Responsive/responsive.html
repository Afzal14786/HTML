Position Property

  • Static (Default) : "Elements follows the normal document flow. (top, right, bottom,left, z-index wound not work)"
  • Relative : "Elements positions adjusted from it's normal position."
  • Absolute : "Positions elements relative to the nearest positioned ancestor."
  • Fixed : "Element positioned relative to the viewport, does not move on scroll."
